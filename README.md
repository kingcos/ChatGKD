# ChatGKD

A new app for ChatGPT in SwiftUI.

> What is *ChatGKD*?
>
> GKD stands for Gao（搞）Kuai（快）Dian（点）, which means faster.

## Usages

Just replace your API key in `Utils/ChatGPTHelper.swift`:

```swift
_api = ChatGPTAPI(apiKey: "YOUR_API_KEY")
```

Build, and running.
